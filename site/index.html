Quick Start

Up and running in minutes with Docker Compose.

1. Clone and start services

bash
git clone https://github.com/techsavvyash/heimdall.git
cd heimdall
docker compose up -d

2. Run migrations and start the server

bash
make migrate
make run

3. Register your first user

bash
curl -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "SecurePass123!",
    "firstName": "Jane",
    "lastName": "Doe"
  }'

4. Install the JS SDK and authenticate

typescript
import { HeimdallClient } from '@heimdall/sdk';

const heimdall = new HeimdallClient({
  apiUrl: 'http://localhost:8080',
  tenantId: 'your-tenant-id',
});

const { user, tokens } = await heimdall.auth.login({
  email: 'user@example.com',
  password: 'SecurePass123!',
});

console.log('Logged in as:', user.email);
